Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Usage:

  • https://pepper-shallot.glitch.me/search/latest
  • https://pepper-shallot.glitch.me/search?term=monkey&offset=10

Notes:

  • Endpoint /search/latest shows last 10 search queries, format is: [{"term":searchterm,"timestamp":ISO_format_date_time_string}, ...]
  • Endpoint /search?term='search string'&offset='start index (optional)' returns JSON in format: [{"url":'image url',"description":'image description',"originalurl":'original url of the image'}, ..]
